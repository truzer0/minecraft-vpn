# Установка модов и шейдеров для Nexus Minecraft

## Автоматическая установка (рекомендуется)

1. Скачай готовый модпак: https://85.215.132.23:8080/modpack/nexus-modpack.tar.gz
2. Распакуй архив в папку `.minecraft`
3. Запусти TLauncher с версией Forge 1.20.4
4. Подключись к серверу: `85.215.132.23`

## Ручная установка

1. Установи TLauncher: https://tlauncher.org/installer
2. Выбери версию Forge 1.20.4
3. Скачай моды индивидуально из папки `/mods`
4. Скачай шейдеры из папки `/shaders`
5. Скопируй всё в соответствующие папки `.minecraft`

## Активация шейдеров

1. Запусти игру
2. Зайди в настройки (Options)
3. Video Settings → Shader Packs
4. Выбери BSL или Complementary
5. Настрой качество под своё железо

## Системные требования

### Минимальные (без шейдеров):
- CPU: 4 ядра, 2.5 GHz
- RAM: 4GB
- GPU: Intel HD Graphics 620

### Рекомендуемые (с шейдерами):
- CPU: 6 ядер, 3.5 GHz
- RAM: 8GB
- GPU: NVIDIA GTX 1060 / AMD RX 580
